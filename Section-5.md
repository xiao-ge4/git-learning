# Section 5

## 学习目标

分支管理

## 相关指令

| 命令                         | 作用                     | 备注                   |
| ---------------------------- | ------------------------ | ---------------------- |
| git branch (branchname)      | 创建分支                 |                        |
| git checkout (branchname)    | 切换分支命令             |                        |
| git merge                    | 合并分支命令             | 合并完后就可以删除分支 |
| git checkout -b (branchname) | 创建新分支并切换到该分支 |                        |
| git branch -d (branchname)   | 删除分支                 |                        |
