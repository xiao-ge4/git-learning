# Section 6

## 学习目标

标签，记录重要节点

## 相关指令

| 命令                                     | 作用         | 备注 |
| ---------------------------------------- | ------------ | ---- |
| git tag                                  | 查看标签     |      |
| git tag <version>                        | 直接打标签   |      |
| git tag -a <version>                     | 打带注解标签 |      |
| git log --decorate                       | 查看标签     |      |
| git tag -a v0.9 <版本号>                 | 追加标签     |      |
| git tag -a <tagname> -m "runoob.com标签" | 指定标签信息 |      |
